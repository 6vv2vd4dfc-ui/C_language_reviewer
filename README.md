简介
一个用于C语言理论课程复习的答题系统，包含分支循环数据表达式、结构宏、数组和指针四大模块的135道选择题。适合期末考试前复习（yu xi）。

下载与安装
目前仅有离线版本，正在尝试研究网页版。

文件说明

main.c - 主程序文件
quizsource1.h – 目前有整理老师发的期末复习习题得到的135道单项选择题，后续（或许）会补充完善填空题等相关模块

使用指南
在编译器中打开，编译运行。

程序启动后会自动显示欢迎界面和菜单：


===== C语言理论作业答题系统 =====
1. 回答选择题
2. 回答填空题（zhe shi jia de）
3. 查看答题结果
4. 退出系统
答题流程
选择题目类别：目前系统仅开发了选择题功能（选项1），包含：

分支循环数据表达式（64题）：基础语法、运算符、控制结构

结构宏（15题）：预处理指令、结构体、宏定义

数组（28题）：一维/二维数组、字符数组、字符串

指针（28题）：指针基础、指针运算、指针与数组

开始答题：

程序会按顺序显示题目

每个题目显示题目描述和四个选项

输入A、B、C、D选择答案，输入Q退出答题

或者输入T跳转至相应题目

答题后立即显示正确与否，并给出正确答案

查看结果：答题结束后可查看统计结果
